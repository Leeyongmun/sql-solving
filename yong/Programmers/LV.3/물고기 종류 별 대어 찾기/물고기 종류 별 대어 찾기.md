# 물고기 종류 별 대어 찾기

## 문제 요약

물고기 종류별로 가장 큰 물고기의 `ID`, `FISH_NAME`, `LENGTH`를 조회한다.

- 물고기 정보는 `FISH_INFO` 테이블에 있다.
- 물고기 이름은 `FISH_NAME_INFO` 테이블에 있다.
- 같은 물고기 종류 안에서 길이가 가장 큰 물고기를 찾는다.
- 결과는 `ID` 기준 오름차순으로 정렬한다.

## SQL

```sql
SELECT ID, FISH_NAME, LENGTH
FROM (
        SELECT I.ID, N.FISH_NAME, I.LENGTH, ROW_NUMBER() OVER (
                PARTITION BY
                    I.FISH_TYPE
                ORDER BY I.LENGTH DESC
            ) AS RN
        FROM FISH_INFO I
            JOIN FISH_NAME_INFO N ON I.FISH_TYPE = N.FISH_TYPE
    ) T
WHERE
    RN = 1
ORDER BY ID;
```

## 풀이 설명

`FISH_INFO`와 `FISH_NAME_INFO`를 `FISH_TYPE` 기준으로 조인해서 물고기 이름을 함께 조회한다.

```sql
FROM FISH_INFO I
    JOIN FISH_NAME_INFO N ON I.FISH_TYPE = N.FISH_TYPE
```

그다음 `ROW_NUMBER()` 윈도우 함수를 사용해서 물고기 종류별 순위를 매긴다.

```sql
ROW_NUMBER() OVER (
    PARTITION BY I.FISH_TYPE
    ORDER BY I.LENGTH DESC
) AS RN
```

- `PARTITION BY I.FISH_TYPE`: 물고기 종류별로 그룹을 나눈다.
- `ORDER BY I.LENGTH DESC`: 각 종류 안에서 길이가 긴 물고기부터 정렬한다.
- `ROW_NUMBER()`: 정렬된 결과에 1번부터 순위를 붙인다.

따라서 각 물고기 종류에서 가장 큰 물고기는 `RN = 1`이 된다.

```sql
WHERE RN = 1
```

마지막으로 `ORDER BY ID`를 통해 결과를 `ID` 기준 오름차순으로 정렬한다.

## 핵심 포인트

이 문제는 종류별 최댓값을 구한 뒤 해당 행을 찾아야 한다. `ROW_NUMBER()`를 사용하면 `FISH_TYPE`별로 길이 순위를 매길 수 있으므로, `RN = 1` 조건만으로 각 종류의 대어를 선택할 수 있다.
